🌿 AQI Monitoring Android App

An Android application that provides real-time Air Quality Index (AQI) data with custom visual graphs and a smooth, user-friendly experience.

🚀 Features

📡 Live AQI Data  
Fetches real-time AQI data from API and updates dynamically.

📊 Custom Graphs & Charts  
Implements custom-built graphs to visualize AQI trends in an intuitive and engaging way.

🎨 Modern UI (Material Design)  
Clean and responsive UI built using Material Design principles.

🌙 Dark Mode Support  
Supports night mode for improved usability in low-light environments.

📍 Location-Based AQI  
Displays AQI based on the user’s current location.

⚡ Efficient Data Handling  
Optimized API calls and UI updates for smooth performance.

🛠️ Tech Stack

• Language: Java / Kotlin  
• Networking: Retrofit  
• UI: XML, Material Components  
• Charts: Custom Canvas-based graphs
